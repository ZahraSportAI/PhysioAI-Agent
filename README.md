# PhysioAI-Agent

اولین دستیار هوشمند عامل‌محور (Agentic Workflow) در حوزه فیزیولوژی ورزشی ایران

توسعه‌دهنده: [زهرا احمدی]
زمینه تخصصی: فیزیولوژی ورزش(کارشناس ارشد دانشگاه الزهرا) + مهندسی Agentic AI

چشم‌انداز پروژه:
ساخت سیستمی که با دریافت داده‌های فیزیولوژیک (ضربان قلب، لاکتات، VO2max)، برنامه‌های تمرینی و توانبخشی شخصی‌سازی‌شده ارائه دهد.

وضعیت پروژه: 🟢 در حال توسعه (شروع: تیر ۱۴۰۵)

ابزارهای کلیدی: LangChain, RAG, FastAPI, Docker
